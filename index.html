php-build

php-build builds multiple PHP versions from source, and this fully automatic.

Who uses this?

  • Travis CI
  • Maybe you?

Install

% git clone git://github.com/php-build/php-build
% cd php-build
% sudo ./install.sh

Usage

Synopsis

php-build --help
php-build --definitions
php-build [-i <environment_or_inifile>] <definition> <prefix>

Before you can build PHP you will need all necessary build dependencies.

If you use Ubuntu:

% sudo apt-get install libmcrypt-dev libreadline-dev

% sudo apt-get build-dep php5-cli

If you use macOS:

  1. Install Homebrew

  2. Install Hombrew packages:

    % brew install pkg-config autoconf openssl icu4c re2c bison libzip mcrypt

  3. Specify the Homebrew library places in configure options:

    env "PATH=/usr/local/opt/bison/bin:$PATH" "PHP_BUILD_CONFIGURE_OPTS=--with-openssl=$(brew --prefix openssl) --with-libzip" phpenv install 7.0.30

Then run this to install PHP 5.5.9 in development configuration:

% php-build -i development 5.5.9 ~/local/php/5.5.9

To get a list of all supported PHP versions pass the --definitions flag:

% php-build --definitions
